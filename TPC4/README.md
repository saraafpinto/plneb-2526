Este projeto consiste num Landing Page moderna e responsiva desenvolvida para a Academia de Patinagem de Braga. O objetivo principal é apresentar o clube, as suas modalidades, conquistas e facilitar o contacto de novos atletas interessados em aulas de iniciação.

Funcionalidades Principais
- Design Responsivo: Adaptado para dispositivos móveis utilizando media queries.
- Menu de Navegação Fixo: Facilita o acesso às diferentes secções da página durante o scroll.
- Slider de Notícias Dinâmico: Carrossel interativo implementado com JavaScript para exibir as últimas novidades e competições do clube.
- Integração com Redes Sociais: Links diretos para Instagram, Facebook e um botão de contacto rápido via WhatsApp.
- Secção de Conquistas: Destaque visual para os títulos nacionais e medalhas mundiais da academia.
- Informação de Iniciação: Calendário de horários e sistema de empréstimo de equipamento para novos praticantes.

Tecnologias Utilizadas
- HTML: Estruturação semântica do conteúdo.
- CSS: Estilização personalizada, layouts em Flexbox, animações de hover e transições suaves.
- JavaScript: Lógica para o funcionamento do slider de notícias (scroll horizontal controlado por botões).
- Font Awesome: Biblioteca de ícones para redes sociais e informações de contacto.

Estrutura do Projeto
A página está organizada nas seguintes secções:
- Header: Logo e título da academia.
- Quem Somos: História do clube e público-alvo.
- Modalidades: Detalhes sobre Iniciação, Patinagem Livre, Solo Dance e Show/Precisão.
- Braga Roller Team: Espaço dedicado à equipa de elite e alta competição.
- Notícias: Feed visual com links externos para artigos desportivos.
- Contactos: Localização (integração com Google Maps) e canais de comunicação.
