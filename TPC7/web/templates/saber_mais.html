Sobre o MedDic

Explorando a interseção entre Linguística Computacional e Medicina.

O Projeto

Este dicionário médico digital foi desenvolvido no âmbito da unidade curricular de Processamento de Linguagem Natural (PLN), inserida no 1º ano do Mestrado.

O objetivo principal é a estruturação e visualização de dados extraídos de fontes médicas reais, permitindo uma navegação rápida, intuitiva e tecnicamente rigorosa sobre terminologia clínica.

Como funciona?

  • 1 Extração: Os dados são lidos a partir de um ficheiro JSON estruturado.
  • 2 Processamento: O backend em Python organiza os termos alfabeticamente e limpa a informação.
  • 3 Interface: O Flask serve os dados para esta interface dinâmica e responsiva.

Tecnologias Usadas

Python 3.12
Lógica de processamento e manipulação de ficheiros.
Flask
Micro-framework para a gestão de rotas e servidor web.
Bootstrap 5
Design moderno e componentes responsivos.
